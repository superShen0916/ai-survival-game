- ✅ 改为季度周期：一个季度=90天

## 🎯 游戏核心机制

### 1. 时间系统

```
年度循环：季度1 → 季度2 → 季度3 → 季度4 → 年度总结
季度循环：日常事件 → 中期反馈 → OKR评估 → 结果调整
```

### 2. 隐藏分系统

```
隐藏分 = 技术匹配度 × 0.4 + 团队贡献 × 0.3 + AI适应力 × 0.2 + 市场竞争力 × 0.1
```

### 3. 替代预警机制

```
隐藏分 ≥ 70：安全
50 ≤ 隐藏分 < 70：注意提升
30 ≤ 隐藏分 < 50：替代预警
隐藏分 < 30：被替代风险极高
```

### 4. 绩效评估周期

```
季度评估：每个季度结束（90天）
年度评估：每个年度结束（365天）
```

## 📝 设计说明

### 核心设计理念

1. **真实感**：还原职场的真实运作机制
2. **策略性**：玩家需要做出长远的职业规划
3. **渐进性**：游戏难度随时间逐渐提升
4. **教育性**：传递AI时代的职场生存技巧

### 玩法创新点

1. **隐藏分系统**：模拟真实职场中难以量化的因素
2. **延迟替代**：替代不是立即发生，而是有一个过程
3. **绩效周期**：模拟现实中的季度和年度评估
4. **多结局**：根据不同的选择路径产生不同结局

### 技术实现考虑

- 使用JSON配置文件存储事件和规则
- 实现模块化的系统设计
- 保留数据导出和导入功能
- 支持未来扩展到更多角色类型
