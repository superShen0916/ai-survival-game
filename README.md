# AI替代危机：职场生存战

🎮 一个模拟AI浪潮下程序员职场生存的文字冒险游戏

## 游戏介绍

AI浪潮席卷互联网行业，你的岗位随时可能被AI替代。你能坚持多久不被优化？

这是一个纯前端文字冒险游戏，**打开就能玩**，不需要安装任何东西。

- 真实职场职级体系：P5 → P6 → P7 → M1
- 隐藏分系统：技术匹配度 × 团队贡献 × AI适应力 × 市场竞争力
- **不进则退机制**：AI在进步，每季度所有属性自动衰减
- 渐进难度：年份越久，替代概率越高，三年生存极其困难
- 季度OKR → 半年评估 → 年度评估 的真实绩效周期
- 30+ 职场事件，5个彩蛋事件，P6+ 解锁高级管理/晋升事件
- 多种结局，根据你的选择和属性走向不同结果

## 🎮 立即开始玩

[点击这里直接开始游戏](./simple-game.html)

如果你想在线玩，可以部署到 GitHub Pages 或者任何静态网页服务器。

## 🎯 胜利条件

- 短期：坚持生存 1 个季度（90天）
- 中期：坚持生存 1 年（365天），晋升到 P6
- 长期：坚持生存 3 年（1095天），成为技术专家/管理者

## 📊 核心机制

```
隐藏分 = 技术匹配度 × 0.4 + 团队贡献 × 0.3 + AI适应力 × 0.2 + 市场竞争力 × 0.1

隐藏分 ≥ 70：安全
50 ≤ 隐藏分 < 70：注意提升
30 ≤ 隐藏分 < 50：替代预警
隐藏分 < 30：被替代风险极高

直接淘汰：累计 4 次警告 或者 隐藏分 < 30
难度递增：每年替代概率 +8%，每季度全属性衰减 2 点
```

## 🎭 结局种类

失败结局根据你的属性随机给出不同原因，包括：

- 💀 你被优化了，公司放你长假去寻找诗和远方
- 🏆 技术专家结局 - 深耕技术，AI永远无法替代你
- 👔 技术管理者结局 - 转型管理，领导力无法被AI替代
- 🚀 创业成功结局 - 抓住AI风口，成功创业
- 🌍 自由职业结局 - 离开大厂，活出另一种可能
